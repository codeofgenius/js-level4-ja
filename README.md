# JavaScriptレベル4

JavaScript (JS) の基礎的な学習を修了した方に向けて、より発展的な文法を紹介します。主要な組み込みオブジェクトを取り扱います。

## 組み込みオブジェクト
JavaScriptの組み込みオブジェクトとは、開発者が一から作成しなくても、ブラウザやNode.jsなどの実行環境に最初から備わっている標準的なツール群のことです。

データの基本構造を支える`Object`、複数のデータを保持する`Array`、文字列を扱う`String`、数学計算を行う`Math`、日時の操作を担う`Date`などがあります。これらを活用することで、複雑な処理を簡潔かつ効率的に記述できるのが大きな特徴です。

## 内容
JavaScriptレベル4では以下のテーマを取り扱います。

* Array
* String
* Object
* Map
* Set
* Date
* Math

「ECMAScript2015」以降に準拠していますが、本カリキュラムでは上記の分野のみを取り扱います。
